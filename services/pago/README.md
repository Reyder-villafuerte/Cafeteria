# 💳 Pago Service - Cafetería Microservices

Servicio encargado de gestionar los pagos de pedidos en la cafetería.

---

## 🚀 Tecnologías

- Java 17
- Spring Boot
- Spring Web
- Spring Data JPA
- MySQL
- Lombok
- Swagger (OpenAPI)
- Docker
- Maven

---

## 📦 Funcionalidades

- Crear pago de un pedido
- Consultar pagos
- Actualizar pago
- Eliminar pago
- Integración con pedido-service (cambio de estado)
- Soporte para métodos de pago:
  - EFECTIVO
  - YAPE
